Descrição Geral

O banco de dados registra todos os eventos que influenciam o momento ideal da oferta. A tabela Sessao_Jogo guarda o contexto da partida, enquanto Oferta define os pacotes disponíveis. Cada vez que uma oferta aparece, o registro vai para Disparo_Oferta, permitindo analisar o comportamento do jogador ao longo do tempo.
